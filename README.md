# Labs

Exercicios das aulas praticas e laboratorios de algumas cadeiras do curso LEIC-A.

## IAED (C)
Introducao a Algoritmos e Estruturas de Dados. Codigos em C desenvolvidos nos laboratorios, abrangendo listas, filas, pilhas, grafos e outras estruturas fundamentais.

## LP (Prolog)
Logica e Programacao. Exercicios em Prolog focados em raciocinio logico, manipulacao de listas, recursao e resolucao de problemas com regras e factos.
