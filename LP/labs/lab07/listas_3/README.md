Usando funcionais, defina o predicado `numOcc/3`, tal que `numOcc(L, E, N)`
significa que o elemento `E` ocorre `N` vezes na lista `L`. Por exemplo:

```
?- L = [1,2,3,4,1,5,1], numOcc(L, 1, N).
L = [1, 2, 3, 4, 1, 5, 1],
N = 3.
?- L = [1,2,3,4,1,5,1], numOcc(L, 11, N).
L = [1, 2, 3, 4, 1, 5, 1],
N = 0.
```
