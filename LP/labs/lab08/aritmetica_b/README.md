Defina o predicado `inverte/2`, tal que `inverte(N, Inv)`, em que `N` é um inteiro positivo, significa que `Inv` é o resultado de inverter os dígitos de `N`. Por exemplo:

```
?- inverte(123, Inv).
Inv = 321 .
```
