Defina o predicado `somaDigitos/2`, tal que `somaDigitos(N, S)`, em que `N` é um inteiro positivo, significa que `S` é a soma dos dígitos de `N`. Por exemplo:

```
?- somaDigitos(123, S).
S=6
```

Gerando um processo recursivo.
